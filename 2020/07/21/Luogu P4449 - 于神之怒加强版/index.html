Luogu P4449 - 于神之怒加强版

描述

给定 n,m,kn,m,kn,m,k，计算

(∑i=1n∑j=1mgcd⁡(i,j)k) mod (109+7)\left(\sum_{i=1}^n\sum_{j=1}^m\gcd(i,j)^k\right)\bmod (10^9+7) (i=1∑n​j=1∑m​gcd(i,j)k)mod(109+7)

TTT 组询问。

1≤T≤2×1031\leq T\leq 2\times 10^31≤T≤2×103，1≤n,m≤5×1061\leq n,m\leq 5\times 10^61≤n,m≤5×106

分析

令 n<mn<mn<m

\begin{align} &\sum_{i=1}^n\sum_{j=1}^m\gcd(i,j)^k\\ =&\sum_{i=1}^n\sum_{j=1}^m\sum_{d=1}^n[\gcd(i,j)=d]d^k\\ =&\sum_{d=1}^nd^k\sum_{i=1}^{\left \lfloor \frac n d\right \rfloor}\sum_{j=1}^{\left \lfloor \frac m d\right \rfloor}[\gcd(i,j)=1]\\ =&\sum_{d=1}^nd^k\sum_{i=1}^{\left \lfloor \frac n d\right \rfloor}\sum_{j=1}^{\left \lfloor \frac m d\right \rfloor}\sum_{x\mid \gcd(i,j)}\mu (x)\\ =&\sum_{d=1}^nd^k\sum_{x=1}^{\left \lfloor \frac n d\right \rfloor}\sum_{i=1}^{\left \lfloor \frac n {dx}\right \rfloor}\sum_{j=1}^{\left \lfloor \frac m {dx}\right \rfloor}\mu (x)\\ =&\sum_{d=1}^nd^k\sum_{x=1}^{\left \lfloor \frac n d\right \rfloor}\left \lfloor \frac n {dx}\right \rfloor \left \lfloor \frac m {dx}\right \rfloor\mu (x)\\ \end{align}

设 T=dxT=dxT=dx，得到

∑T=1n⌊nT⌋⌊mT⌋∑d∣Tdkμ(Td)\sum_{T=1}^n\left \lfloor \frac n {T }\right \rfloor \left \lfloor \frac m {T}\right \rfloor\sum_{d\mid T}d^k\mu\left(\frac T d\right) T=1∑n​⌊Tn​⌋⌊Tm​⌋d∣T∑​dkμ(dT​)

设 f(T)=∑d∣Tdkμ(Td)\displaystyle f(T) = \sum_{d\mid T}d^k\mu\left(\frac T d\right)f(T)=d∣T∑​dkμ(dT​)。则对于任意互质的 a,ba,ba,b 有

f(a)⋅f(b)=∑i∣aikμ(ai)∑j∣bjkμ(bj)=∑ij∣ab(ij)kμ(abij)=f(ab)f(a)\cdot f(b)=\sum_{i\mid a} i^k\mu\left(\frac a i\right)\sum_{j\mid b} j^k\mu\left(\frac b j\right)=\sum_{ij\mid ab}(ij)^k\mu\left(\frac{ab}{ij}\right)=f(ab) f(a)⋅f(b)=i∣a∑​ikμ(ia​)j∣b∑​jkμ(jb​)=ij∣ab∑​(ij)kμ(ijab​)=f(ab)

故 fff 为积性函数。且对任意质数 ppp 有

\left\{\begin{align}&f(p)=p^k-1\\ &f\left(p^{k}\right) = p^{k(k-1)}\left(p^k-1\right)\\&f\left(p^{k+1}\right)=p^{k^2}\left(p^k-1\right)=p^kf\left(p^k\right)\end{align}\right.

这时便可线性筛预处理出 fff。

回到上式 ∑T=1n⌊nT⌋⌊mT⌋f(T)\displaystyle \sum_{T=1}^n\left \lfloor \frac n {T }\right \rfloor \left \lfloor \frac m {T}\right \rfloor f(T)T=1∑n​⌊Tn​⌋⌊Tm​⌋f(T)，数论分块一下就好了。
